# Billboard Promo — Phase 5 Plan

## Phase 5: Offer and Value — Show what the user gets

### Goal
Turn the proof into a simple offer. The viewer should understand what Billboard gives them in clear, concise terms.

### Core idea
The video should now move from showing the product to explaining the value in a compact and intelligent way.

### Visual concept
A premium value card or stacked list appears centered. Each item enters clearly and feels deliberate.

### What should appear on screen
Use a short list of high-value benefits, such as:
- 30-day broadcast rotation
- permanent archive
- priority or amplification
- analytics or visibility
- template production

Keep it to a small number of items so it stays clean.

### Motion language
The motion should feel polished and confident.
Use:
- one item at a time
- subtle stagger
- a controlled card or list reveal
- clean spacing and strong contrast

The scene should feel like a concise offer, not a feature dump.

### Timing
Suggested timing:
- 0:17.5–0:23.5

This phase should feel like the “what you get” section of the video.

### Design notes
- Use a compact premium card or stacked layout
- Keep the language short
- Do not overfill the frame
- Make the value feel tangible and useful

### Typography and UI cues
- Use bold but readable type
- Keep the list left-aligned or centered, depending on the layout
- Use orange accents sparingly for the strongest benefit

### Motion notes for implementation
Use a simple rhythm:
1. Value card appears
2. Benefit lines reveal in sequence
3. One strong benefit is emphasized
4. Hold briefly so the viewer can absorb it

### What this phase must achieve
By the end of this phase, the viewer should understand:
- Billboard gives more than exposure
- the offer is clear
- the value is tangible

### Implementation guidance for Cursor
Treat this as a clarity scene.
Keep the copy short and the visuals premium.
Do not try to say everything at once.

### Cursor prompt
Create a value scene for the Billboard promo that turns the product proof into a simple and premium offer. Show a compact feature card or stacked list of benefits, revealing them one by one in a calm and elegant rhythm. Keep the list concise and direct so the value feels clear and persuasive.

### Success criteria
- The value is immediately understandable
- The scene feels premium and controlled
- The list does not feel like a wall of text
- The next phase can move into the objection-removal beat naturally
