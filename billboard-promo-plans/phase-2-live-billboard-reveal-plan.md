# Billboard Promo — Phase 2 Plan

## Phase 2: Live Billboard Reveal — The product comes alive

### Goal
Immediately pivot from the hook into the first real proof of the product. The viewer should understand that Billboard is not an abstract idea; it is a live, active broadcast surface.

### Core idea
The moment after the hook should feel like the screen is waking up. The dark void opens into a premium live interface and the product starts to feel real.

### Visual concept
Reveal a polished Billboard surface with:
- the on-air header
- the live telemetry strip
- the player frame
- a glowing orange broadcast accent
- a subtle broadcast atmosphere

This is the first “this exists” moment.

### What should appear on screen
Use the product surface as the hero:
- top-left on-air header
- telemetry strip with live status
- player frame with a rotating halo or subtle conic glow
- small product card or “NOW BROADCASTING” info block
- a faint live-state atmosphere around the player

### Motion language
The transition should feel cinematic and confident.
Use:
- a dark-to-light reveal
- a slight zoom-in or push-in into the surface
- soft glow-on-entry
- clean UI emergence
- no clutter

The motion should feel like the product is turning on.

### Timing
Suggested timing:
- 0:02.5–0:07.5

This is a strong mid-early reveal phase and should feel like the first real payoff.

### Design notes
- Background remains dark and premium
- The player is the hero object
- Orange accents should be used for status and emphasis
- Keep the interface crisp and readable
- Avoid overdecorating the screen

### Typography and UI cues
- Headline is not needed yet; let the UI carry the message
- Use mono styling for telemetry and labels
- Keep the live-state elements clean and readable
- The product card should feel like an actual broadcast overlay

### Motion notes for implementation
Use a layered reveal:
1. Dark background stays present
2. Broadcast surface fades in with controlled opacity
3. Telemetry strip enters cleanly
4. Player frame emerges with a glow
5. Subtle push-in brings the product forward

The transition should feel like a signal turning on, not a slide.

### What this phase must achieve
By the end of this phase, the viewer should feel:
- this product is real
- it looks premium
- it has active state
- it is not just a static mockup

### Implementation guidance for Cursor
Treat this as a product reveal scene.
The UI should feel precise and grounded.
Do not overcomplicate the scene with extra copy.
The product surface is the story.

### Cursor prompt
Create the first product reveal for the Billboard promo. Transition from the dark hook into a premium live Billboard interface with an on-air header, telemetry strip, glowing player frame, and a subtle live broadcast atmosphere. Make it feel like the product is turning on in a cinematic way. Use strong orange accents and a polished premium look without adding clutter.

### Success criteria
- The reveal feels cinematic and premium
- The product surface is visually clear
- The viewer understands this is a live broadcast environment
- The scene transitions naturally into the next proof beat
